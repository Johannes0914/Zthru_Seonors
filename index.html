Multi-Sensor Logger and Gateway Using BLE

Project Overview

This project focuses on developing a multi-sensor logger system using Bluetooth Low Energy (BLE) for real-time environmental monitoring. The system collects data such as temperature and humidity, transmitting it to a gateway for aggregation and visualization.
